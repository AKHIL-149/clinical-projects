## Getting Started

### Prerequisites

```bash
# SAS (University Edition or Licensed)
# R and RStudio (>= 4.0.0)
# Python (>= 3.8)
```

### Installation

```bash
git clone https://github.com/AKHIL-149/clinical-projects.git
cd clinical-projects
```
